Over de Maker

Wie ben ik?

Hallo! Mijn naam is Jan en ik ben een student aan Avans. Deze website is gemaakt als onderdeel van mijn webdesign cursus.

Ik heb altijd al een grote passie gehad voor films. Van klassieke cinema tot moderne blockbusters, ik geniet van alle aspecten van filmkunst - van scenario's en regie tot cinematografie en soundtracks. Het creëren van deze website heeft me de mogelijkheid gegeven om mijn twee interesses te combineren: film en webontwikkeling.

Mijn Opleiding

Ik volg momenteel een opleiding in webontwikkeling waarbij ik leer werken met moderne technologieën zoals HTML5, CSS3, JavaScript en frameworks zoals Bootstrap. Deze website is een demonstratie van de vaardigheden die ik tot nu toe heb opgedaan.

Over Dit Project

Movie Explorer is een responsive website gebouwd met Bootstrap 5 die dient als een platform voor filminformatie. Het project omvat verschillende pagina's met complexe layouts, formulieren, en interactieve elementen.

De site bevat de volgende functies:

  • Een responsive navigatiemenu
  • Een automatische carrousel met filmposters
  • Een grid layout met 12 verschillende films
  • Gedetailleerde filmpagina's met uitklapbare beschrijvingen
  • Acteur profiel pagina's
  • Een formulier voor het toevoegen van nieuwe films
  • Een zoekfunctie voor films
  • Review systeem met ratings

Over dit project

Deze website is gemaakt als schoolopdracht voor de cursus Webontwikkeling. Alle film informatie en afbeeldingen zijn gebruikt voor educatieve doeleinden. De website demonstreert het gebruik van Bootstrap voor responsive webdesign, CSS voor custom styling, en JavaScript voor interactieve functionaliteit.
